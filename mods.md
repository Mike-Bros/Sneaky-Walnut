## 🔧 Server Tools & Utilities

Enhance server management, improve performance, and provide essential admin commands.

### 📦 Libraries & Core APIs

Essential libraries required by multiple mods.

- [Architectury API](https://modrinth.com/mod/architectury) – Client & server intermediary API for multiplatform mod development.
- [BCLib](https://modrinth.com/mod/bclib) – Core library providing essential utilities and world generation support for mods like BetterEnd and BetterNether.
- [Cardinal Components API](https://modrinth.com/mod/cardinal-components-api) – Components API for Fabric & Quilt.
- [Cristel Lib](https://modrinth.com/mod/cristel-lib) – Utility library offering shared code and tools required by various mods for enhanced compatibility and functionality.
- [Collective](https://modrinth.com/mod/collective) – Shared code library to improve mod compatibility.
- [Fabric API](https://modrinth.com/mod/fabric-api) – Core library for most Fabric mods.
- [Fabric Language Kotlin](https://modrinth.com/mod/fabric-language-kotlin) – Enables Kotlin language support.
- [Geckolib](https://modrinth.com/mod/geckolib) – Animation library used by several mods.
- [Indium](https://modrinth.com/mod/indium) – Provides Sodium compatibility with the Fabric rendering API (FRAPI), enabling mod features like connected textures and custom models.
- [Puzzles Lib](https://modrinth.com/mod/puzzles-lib) – Library for utility features and mod support.
- [Resourceful Lib](https://modrinth.com/mod/resourceful-lib) – Core library providing essential utilities and support for various mods.
- [Sophisticated Core (Unofficial Fabric port)](https://modrinth.com/mod/sophisticated-core-(unofficial-fabric-port)) – Required for Sophisticated Backpacks.

### 🛡️ Admin & Command Utilities

Manage commands and player actions efficiently.

- [Compact Help Command](https://modrinth.com/mod/compact-help-command) – Paginated and organized `/help` command.
- [Essential Commands](https://modrinth.com/mod/essential-commands) – Teleportation, home setting, and permissions commands.
- [Fast Backups](https://modrinth.com/mod/fastback?version=1.20.1&loader=fabric) - Incremental world backups.
- [Ledger](https://modrinth.com/mod/ledger) – Server-side logging and activity tracking.
- [Trade Cycling](https://modrinth.com/mod/trade-cycling) – Allows players to cycle through villager trades easily.

### ⚙️ Performance & World Management

Improve world generation, chunk loading, and overall server performance.

- [Carpet](https://modrinth.com/mod/carpet) – Custom server tweaks and performance settings.
- [Chunky](https://modrinth.com/mod/chunky) – Pre-generates chunks to reduce lag.
- [Hold That Chunk](https://modrinth.com/mod/hold-that-chunk) – Keeps specified chunks loaded without the need for constant redstone or player presence, improving automation and farms.
- [Sodium](https://modrinth.com/mod/sodium) – Optimizes rendering performance for higher frame rates and improved visual stability without sacrificing visual quality.

### 👥 Player & Villager Behavior Tweaks
